bbPress GDPR helps website owners to comply with European privacy regulations (GDPR).

== Description ==

The GDPR requires companies and site owners to be transparent about how they collect, use, and share personal data. It also gives individuals more access and choice when it comes to how their own personal data is collected, used, and shared.

bbPress GDPR plugin extends new Data Export & Data Erasure feature bundled with WordPress 4.9.6 Release.

This plugin covers all bbPress native components.

1. Forums (Delete/Export on user requests)
2. Topics (Delete/Export on user requests)
3. Replies (Delete/Export on user requests)

NOTE:- Activating this plugin doesn't guarantee you fully comply with GDPR. Please, contact a GDPR law consultant.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'bbPress GDPR '
3. Activate bbPress GDPR Search from your Plugins page.

= From WordPress.org =

1. Download bbPress GDPR.
2. Upload the 'bbpress-gdpr' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, etc...)
3. Activate bbPress GDPR from your Plugins page.
